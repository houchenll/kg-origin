## 用途
1. 执行 cypher 命令
2. 查看结果

## 常用命令
```
:clear    清空result frame
:guide intro    Browser向导
:guide concepts    图模型概念
:guide cypher    cyper基本用法
:guide movie-graph    使用movie和cypher查询和推荐示例
:guide northwind-graph    把关系数据转换并导入到图谱
:help    帮助
:history    查看历史命令
:play start    查看向导
:server status    connection status
```
